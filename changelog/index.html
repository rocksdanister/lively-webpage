DD May 2024 | Lively v2.1.0.0 Update

Better Everything

This update has been cooking for months.. lots of features and fixes.

New "Music Tunnel" Wallpaper

Mesmerizing infinite music tunnel powered by shaders and Lively API with upto 7 customisation options.

Preview: https://youtube.com/watch?v=-0zJkmno8hs

Better Screensavers

Now you can apply screensaver different from running wallpaper!

Even better Screensavers can now run without having Lively run in the background.

Requires new Screensaver plugin.

Better Control Panel

New display selector interface that resemble Windows settings.

This one took some work, there is no stock way to do this.. had to create a custom control from scratch.

Better Performance - 100,000+ Controls!

Rewrote the wallpaper customise menu code from scratch for performance and quick maintenance.

Thanks to virtualization and Templating the new system is tested to handle 100,000+ controls easily.

Better Update Experience

App updater (installer) has been revamped and better integrated into the main interface.

Latest changelog can be viewed directly from the app.

Better Wallpaper Scaling

Wallpaper scaler settings is now applied per media file and wallpaper layout like span, duplicate.

This feature is only available when Mpv video player is selected.

For backward compatibility old scaler setting is shown when the selected video player don't support this feature.

Better FolderDropdown Control

Redesigned FolderDropdown control with file preview and deletion support.

Preview is supported for most popular formats as it is generated from system shell.

Patreon Supporters <3

Just a little something to show appreciation.

UI/UX

  • New light themed system-tray menu.
  • Added some transition animations to settings page.
  • Replaced Dropdown with Segmented control in key areas.

Windows 24H2 Support

Lively has been upgraded to handle changes in upcoming version of Windows 11.

ARM64 Support and 64bit Transition

  • Native ARM64 support for upcoming snapdragon devices and original surface line.
  • Starting from this release Lively will slowly start transitioning from 32bit to 64bit for better performance.

Fixes

  • Fixed blank app screen issue due to region format settings in some systems.
  • Fixed close wallpaper button disabled in span layout when non-primary display was selected previously.
  • Fixed video wallpaper position not synchronized on startup in duplicate layout.
  • Fixed Customize wallpaper not applying to selected screen if wallpaper is not running.
  • Fixed application crashing when adding some program to wallpaper pause rules.
  • Fixed high system resource usage during update download.
  • Added Unicode support for LivelyProperties Textbox and FolderDropdown. For example, now you can interact with text and filenames with Chinese characters.
  • When a unique DeviceId is not available for display, a hashcode is generated which should resolve some display-related wallpaper bugs.
  • Optional TaskbarCrashTimeOutDelay settings variable to configure the check time between Taskbar Explorer process crash.
  • Exit Screensaver using touch or keyboard input.

Translation

  • Added Belarusian, Galician, Kazakh, Burmese and Slovak languages.
  • Updated community translations.

Updates

  • Upgraded .NET runtime to 8 (after updating the installer may prompt a restart.)
  • Updated WindowsAppSDK v1.5.240404000
  • Updated Mpv player.
  • Updated CEF player.
  • Updated dependencies.
